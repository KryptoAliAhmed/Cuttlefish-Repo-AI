# Cuttlefish Collective Backend Stack

This backend powers the Cuttlefish Collective's AI agents for regenerative infrastructure.

## 🐙 Features
- TrustGraph memory system
- Ethical Constitution alignment checks
- Multi-agent architecture (BuilderAgent, PermitAgent, etc.)
- FastAPI server with CORS
- Grid-aware and permitting integrations

## 🚀 Quickstart

### 1. Clone Repo & Navigate
```bash
git clone <your-repo-url>
cd cuttlefish-backend
```

### 2. Build and Run with Docker Compose
```bash
docker-compose up --build
```

### 3. Access API
Visit: [http://localhost:8000/docs](http://localhost:8000/docs) for Swagger UI

## 🔧 Environment
- Python 3.11
- Uvicorn
- FastAPI
- NetworkX
- Pydantic
